🗂️ การดึงข้อมูลจากหน่วยความจำ

Memory Card Extraction​

การดึงข้อมูลจากสื่อหน่วยความจำ (Memory Card)

ขั้นตอน​

  1. เลือกประเภทไฟล์ที่ต้องการ
  2. กด NEXT
  3. ใส่สื่อหน่วยความจำลงในช่องเสียบ
Adapter

ขึ้นอยู่กับประเภทของสื่อหน่วยความจำ อาจจำเป็นต้องใช้เครื่องดัดแปลง (Adapter)

ระหว่างการดึงข้อมูล​

ระบบจะทำการ:

  • บีบอัดข้อมูล
  • คำนวณค่า Hash
  • เคลียร์ข้อมูลชั่วคราว

เมื่อเสร็จสิ้น​

  1. หน้าจอแสดงผลจะระบุว่าการอ่านสำเร็จ
  2. ถอดการ์ดหน่วยความจำเพื่อดำเนินการต่อ

🔎 การกรองข้อมูล

Data Filtering​

Unicorn มีความสามารถในการกรองข้อมูล:

อุปกรณ์ที่รองรับ​

  • อุปกรณ์ Apple (iOS)
  • อุปกรณ์ Android
  • การ์ดหน่วยความจำแยก
  • USB Mass Storage

การกรองวันที่ (Date Filtering)​

ป้อนช่วงวันที่:

  • วันที่เริ่มต้น (Start Date)
  • วันที่สิ้นสุด (End Date)
tip

สามารถป้อนวันที่โดยแป้นพิมพ์หรือตัวเลือกวันที่บนหน้าจอ

การกรองประเภทไฟล์ (File Type Filtering)​

เลือก/ยกเลิกประเภทไฟล์:

  • รูปภาพ
  • วิดีโอ
  • เสียง
  • เอกสาร
  • แอปพลิเคชัน

การกรองแอป (App Filtering)​

Unicorn จะพยายามระบุไฟล์ที่สามารถถอดรหัสได้:

  • เลือก/ยกเลิกตัวถอดรหัสแอป
  • กดปุ่ม i เพื่อดูรายละเอียด
หมายเหตุสำคัญ

หากพบว่าจำเป็นต้องใช้ไฟล์ที่ถูกกรองออก จะต้องดำเนินการดึงข้อมูลใหม่

ขั้นตอนต่อไป​

  • การส่งออกรายงาน
  • การส่งออกรายงาน
